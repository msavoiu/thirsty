# Thirsty

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Thirsty** is a full-stack web application that helps users locate and contribute information about drinking water sources in their area. Users can mark locations on a map, indicate whether water is hot or cold, and share photos and descriptions. Thirsty aims to promote easy access to clean drinking water and community-sourced knowledge.  

---

## Features

- **Interactive Map:** View and add water source markers with location, description, and photos.  
- **User Authentication:** Secure registration and login with hashed passwords.  
- **Water Source Details:** Indicate hot/cold water availability and add custom notes.  
- **Profile Management:** Users can manage their markers and view their contributions.  
- **Responsive Design:** Fully responsive interface for desktop and mobile users.  

---

## Tech Stack

- **Frontend:** React, Next.js, TypeScript, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** PostgreSQL, Prisma ORM  
- **Authentication:** bcrypt for password hashing, JWT for session management  
- **Maps:** Google Maps JavaScript API with custom markers and info windows  
- **Deployment:** Vercel
